## Tech stack

- Auth: Clerk
- Next 13: App Router && Server Component
- Prisma: Schema
- Plantscale: DB
- Vercel: Deployment && logging && monitoring
- Tailwind ShadCN: Styling
- Cloudiary: Image service
- Strip: Payment

## Running local

`bun install
bun run dev`
